# chinese-wc26umc-guide-f6b3

本仓库用于归档和发布多个独立的 HTML 页面。这些页面各自包含独立的展示内容或功能模块，彼此之间无直接关联。

## 项目简介

这是一个以静态 HTML 页面为主要内容的归档仓库。每个页面均为独立文件，不依赖特定的域名或网站环境，可直接通过浏览器打开查看。

仓库的设计目标是为一些需要长期保存或分发的页面提供统一的存放位置，方便索引与访问。

## 目录结构

```
.
├── README.md          # 本文件
├── index.html         # 仓库入口页面（可选）
├── page-1.html        # 示例归档页面
├── page-2.html
└── ...
```

- `index.html` 为可选入口页，用于展示所有归档页面的链接列表。
- 其他 `.html` 文件为实际归档内容，命名方式尽量反映页面主题。

## 页面归档说明

- 所有页面均为纯前端 HTML，不包含后端依赖。
- 页面资源（如图片、样式、脚本）均已内嵌或同目录存放，确保离线可查看。
- 每个页面独立维护，互不影响。

## 维护说明

- 如需新增页面，请将 HTML 文件放置于仓库根目录，并视情况更新 `index.html` 中的链接。
- 页面内容更新时，请保持文件名不变，以避免链接失效。
- 本仓库不面向特定网站或服务，不提供任何形式的推广或引导。

## 许可

本仓库内容仅供归档与展示用途，具体许可信息请参考各页面内的说明（如有）。
